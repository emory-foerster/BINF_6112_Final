ORF and Frameshift Analysis Toolkit

A bioinformatics tool for scanning genomes, finding ORFs, and flagging possible frameshifts.

FASTA input ORF detection Frameshift candidates HTML, CSV outputs
Home Report Frameshift Plot Files

Upload or provide a FASTA file, then this tool identifies open reading frames (ORFs) across frames 0, 1, 2, picks the dominant ORF, and checks for nearby ORFs that may suggest a frameshift.

At a glance

What it takes in
Nucleotide FASTA file
What it produces
HTML report + CSV summary + JSON Output
Best for
Viral genomes and variants

Quick start

Run the analysis

Example command. Update paths to match your repo: 

python src/main.py -f datasets/input.fasta -oc out_results/results.csv -oh out_results/full_report.html

After it runs, open the report page or the generated HTML report.

What the tool is checking

ORF search
Scan for start and stop codons and record ORFs across reading frames.
Dominance
Compare the longest ORF length to the total ORF signal in the sequence.
Frameshift candidates
Look for strong neighboring ORFs in shifted frames near key positions.
