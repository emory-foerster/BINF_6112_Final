### Code for the function 
```
def parse_codons(seq: str, frame:int) -> list[tuple[str, int]:
    codons = []
    for i in range(frame, len(seq) -2, 3):
        codon = seq[i:i+3]
        if len(codon) == 3:
            codons.append((codon,i))
    return codons
```

### Example input
- parse_codons(seq,frame)
	- seq: "ATGCGTACGTTAGCTAGCTAGCTAGCTATA"
	- frame: 1

### Example output
- list of tuples
[('TGC', 1), ('GTA', 4), ('CGT', 7), ('TAG', 10), ('CTA', 13), ('GCT', 16), ('AGC', 19), ('TAG', 22), ('CTA', 25)]
